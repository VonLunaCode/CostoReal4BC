# CostoReal

Un nuevo proyecto Flutter para la universidad.

## Configuración para el Equipo

Para empezar a trabajar en este proyecto, asegúrate de tener instalado Flutter y sigue estos pasos:

1.  **Clonar el repositorio**:
    ```bash
    git clone <url-del-repositorio>
    ```
2.  **Obtener dependencias**:
    ```bash
    flutter pub get
    ```
3.  **Ejecutar el proyecto**:
    ```bash
    flutter run
    ```

## Estructura del Proyecto

- `lib/`: Contiene el código fuente en Dart.
- `assets/`: (Por crear) Para imágenes, fuentes, etc.
- `test/`: Pruebas unitarias y de integración.

## Convenciones

- Usar nombres de archivos en `snake_case`.
- Mantener los componentes pequeños y reutilizables.
